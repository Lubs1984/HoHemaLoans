## Currency Formatting

All languages use the South African Rand (ZAR) but format differently:

```typescript
// In your i18n config, format function:
if (format === 'currency') {
  return new Intl.NumberFormat(lng, {
    style: 'currency',
    currency: 'ZAR',
    currencyDisplay: 'symbol' // Will show 'R'
  }).format(value);
}
```

**Examples:**
- English: R 10,000.00
- Afrikaans: R 10 000,00
- Most indigenous languages: R 10 000,00 (space-separated thousands)

## Professional Translation Checklist

When sending for professional translation, include:

- [ ] Context about the app (financial services, loans)
- [ ] Target audience (everyday South African users)
- [ ] Tone preference (professional but accessible)
- [ ] Specific terminology standards
- [ ] Cultural sensitivity requirements
- [ ] Sample screenshots
- [ ] Glossary of key terms
- [ ] Request for back-translation verification

## Notes on Language-Specific Formatting

### Click Consonants (isiXhosa, isiZulu, isiNdebele)
- Ensure UTF-8 encoding
- Test font rendering: c, q, x, hl, etc.
- May need web font with full character support

### Tone Marks (Sepedi, Setswana, Sesotho)
- š, ŋ, and other diacritics must display correctly
- Most modern fonts support these

### Long Words
- Tshivenda and Xitsonga can have very long compound words
- Test button and container widths
- May need responsive text sizing
